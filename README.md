# Requirements

Install NodeJs + NPM or Yarn

# Install

1. Clone this project
   ```shell
   git clone https://github.com/sagwascript/quranqu.git
   ```
2. Install dependencies
   ```shell
   cd quranqu
   yarn # npm install
   ```
3. Run
   ```shell
   yarn dev  # npm run dev
   ```
